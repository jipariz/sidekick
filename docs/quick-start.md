# Quick Start

After [installing](installation.md) the core module, wrap your root composable with `SidekickShell` and pass your list of plugins:

```kotlin
@Composable
fun App() {
    val networkPlugin = remember { NetworkMonitorPlugin() }
    val plugins = remember(networkPlugin) { listOf(networkPlugin) }

    MaterialTheme {
        SidekickShell(plugins = plugins) {
            // your app content
        }
    }
}
```

A small FAB appears in the bottom-right corner. Tap it to open the Sidekick panel.

---

## Adding More Plugins

Pass any combination of plugins to `SidekickShell`:

```kotlin
@Composable
fun App() {
    val networkPlugin = remember { NetworkMonitorPlugin() }
    val logPlugin = remember { LogMonitorPlugin() }
    val prefsPlugin = remember { AppPreferencesPlugin() } // generated by KSP

    MaterialTheme {
        SidekickShell(plugins = listOf(networkPlugin, logPlugin, prefsPlugin)) {
            MyAppContent()
        }
    }
}
```

Each plugin becomes a card in the Sidekick panel grid. Tap a card to open that plugin's screen.
